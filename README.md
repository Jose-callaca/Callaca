Creacion de una API REST con python 3 y flask

# Pasos para ejecutar

1. Tener python instalado `version 3.6.*`

2. Instalar flask

`pip3 install flask`

3. Instalar slugify

`pip install awesome-slugify`

4. Correr la aplicacion

`python3 app.py`
